# Реализация обхода графа в ширину (BFS) на Java

## 📌 Описание проекта
Данный проект представляет собой консольное приложение на языке Java, демонстрирующее работу алгоритма **BFS (Breadth-First Search)**. Программа позволяет пользователю интерактивно задать структуру графа, выбрать начальную вершину и получить порядок обхода с расчетом уровней удаленности.

## 🚀 Цели и задачи
* **Реализация графа:** Использование списков смежности (`List<List<Integer>>`) для эффективного хранения связей.
* **Реализация BFS:** Использование очереди (`Queue`) для послойного обхода.
* **Интерактивность:** Динамический ввод данных через консоль (`Scanner`).
* **Дополнительный функционал:** Автоматический расчет кратчайшего расстояния (уровня) от начальной вершины до всех остальных.

## 🛠 Технологии
* **Язык:** Java 11+
* **Среда разработки:** IntelliJ IDEA
* **Библиотеки:** `java.util.*` (Scanner, Queue, LinkedList, ArrayList)

## 📋 Инструкция по запуску
1. Откройте проект в **IntelliJ IDEA**.
2. Убедитесь, что файл называется `Main.java`.
3. Запустите метод `main`.
4. Следуйте инструкциям в консоли:
    * Введите количество вершин (например, `6`).
    * Введите количество связей (ребер).
    * Введите пары вершин, образующих ребро (например, `0 1`).
    * Выберите стартовую точку для обхода.

## 🧠 Логика алгоритма (Контрольные вопросы)

### Как работает BFS?
Алгоритм использует метод «расширяющейся волны». Он посещает все вершины на текущем расстоянии от старта, прежде чем перейти к узлам следующего уровня.

### Чем BFS отличается от DFS?
| Характеристика | BFS (В ширину) | DFS (В глубину) |
| :--- | :--- | :--- |
| **Структура данных** | Очередь (Queue) | Стек (Stack) / Рекурсия |
| **Принцип** | FIFO (First In, First Out) | LIFO (Last In, First Out) |
| **Поиск пути** | Находит кратчайший путь | Не гарантирует кратчайший путь |
| **Слои** | Обход по слоям (уровням) | Идет до "тупика" по одной ветке |

### Почему используется очередь?
Очередь необходима для строгого соблюдения порядка: вершины, обнаруженные раньше, должны быть обработаны первыми. Это гарантирует, что мы не перейдем к уровню $N+1$, пока не посетим все вершины уровня $N$.
